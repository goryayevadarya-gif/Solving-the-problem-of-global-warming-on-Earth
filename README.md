## Определение целевой аудитории и идеи для бота
Чтобы бот был полезным, нужно было написать его понятным для выбранной целевой аудитории.

Целевая аудитория: подростки-взрослые.

Бот: отвечает на заданный вопрос пользователя.

Идеи: у пользователя будет выбор "случайный факт", "точный вопрос", "решение проблемы глобального потепления"/ выбор вопроса у бота/ бот выдавал сайты которые помогли бы решить эту проблему/ бот объяснял чем опасно глобальное потепление и после давал варианты вопросов/ мини игра, бот задает вопросы и если неверно отвечают объяснять и говорить верный ответ.

## Команды нашего бота
Команды:
1. /start - запускает бота
2. /exit_game - останавливает мини-игру
Все остальное - кнопки:
1. Случайный факт - бот выдает случайный факт о глобальном потеплении
2. Решение проблем - бот выдает решния этой проблемы
3. Полезные ссылки - бот выдает сайт о глобальном потепление
4. Начать мини-игру - бот начинает задавать вопросы и варианты ответов, т.е запускается мини-игра
5. О глобальном потеплении - бот объясняет что такое глобальное потепление, а так же рассказывает в каких странах была эта проблема
